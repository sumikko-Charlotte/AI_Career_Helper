# Slogan 图片说明

## 图片放置位置
请将小组设计的 slogan 图片保存为 `slogan-main.png`，并放置在此目录下。

## 完整路径
```
ai-career-helper-frontend/public/images/slogan-main.png
```

## 图片要求
- 文件名：`slogan-main.png`
- 格式：PNG（推荐）或 JPG
- 建议尺寸：宽度 800-1200px，高度自适应
- 背景：建议透明或白色背景，代码已添加白色外发光效果确保在深色背景下清晰可见

## 使用位置
图片将显示在探索页（SloganPage.vue）的主标题位置，替换原有的文字标题。
