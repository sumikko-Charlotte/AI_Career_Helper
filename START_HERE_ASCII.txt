📋 项目概览
═══════════════════════════════════════════════════════════════

✨ 三大新功能：

  1️⃣ 动态头像 🎨
     • DiceBear API 卡通形象
     • 思考（黄色呼吸）→ 回复（绿色脉冲）→ 待机（蓝色静止）
     • 脉冲指示器实时反馈 AI 状态

  2️⃣ 语音输入 🎙️
     • Web Speech API 原生语音识别
     • 支持中文（zh-CN）
     • 点击麦克风按钮自动识别并填入

  3️⃣ 现代聊天界面 💬
     • AI 气泡左侧灰色，用户右侧蓝色
     • 仿 WhatsApp/微信的凹陷圆角设计
     • Loading 三球波浪动画

📞 常见问题
═══════════════════════════════════════════════════════════════

Q1：麦克风按钮为灰显？
A：您的浏览器可能不支持 Web Speech API，使用 Chrome/Edge

Q2：面试官头像不显示？
A：DiceBear API 无法访问，检查网络连接

Q3：聊天无法发送？
A：后端 FastAPI 未启动，确保运行在 http://127.0.0.1:8000

Q4：我应该读哪个文档？
A：根据您的角色在上面选择对应的文档

Q5：怎样进行测试？
A：打开 TESTING_GUIDE.md，按 5 步详细指南执行
