## ✨ 三大新功能

### 1️⃣ 动态头像 🎨
- DiceBear API 卡通头像
- 思考/回复/待机三态动画
- 脉冲指示器实时反馈

**看到效果**：发送消息时，头像从蓝色变黄，3 秒后变绿，最后恢复蓝色

### 2️⃣ 语音输入 🎙️
- 原生 Web Speech API
- 中文语音识别
- 点击按钮自动识别

**使用方法**：点击输入框左侧麦克风 → 对话筒说话 → 文字自动填入

### 3️⃣ 现代聊天界面 💬
- AI 气泡左侧灰色
- 用户气泡右侧蓝色
- Loading 三球波浪动画

**看到效果**：消息以专业的聊天气泡显示，AI 回复中显示动画
