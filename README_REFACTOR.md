## 🎯 核心功能演示

### 功能 1：动态头像

**场景**：用户发送消息
```
1. 用户输入文字并点击"发送"
2. 立即观察：面试官头像开始放大缩小（黄色呼吸）
3. 后端回复消息到达
4. 头像变为绿色，脉冲加快
5. 3 秒后恢复蓝色静止状态
```

### 功能 2：语音输入

**场景**：用户进行语音交互
```
1. 点击输入框左侧的麦克风按钮（🎙️）
2. 按钮变为红色，开始录音
3. 对麦克风说："你能讲一下 React 吗？"
4. 说话完成后系统自动识别
5. 中文文本自动填入输入框
6. 点击"发送"继续对话
```

### 功能 3：现代聊天界面

**特点**：
- 左侧 AI 气泡：浅灰色，带 DiceBear 头像
- 右侧用户气泡：蓝色渐变，显示用户图标
- 自动换行，圆角设计，阴影效果
- 加载中显示三球波浪动画

## 🐛 遇到问题？

### 问题 1：语音按钮不可用
**原因**：浏览器不支持 Web Speech API
**解决**：使用 Chrome/Edge，或检查浏览器设置

### 问题 2：面试官头像不显示
**原因**：DiceBear API 无法访问
**解决**：检查网络连接，或尝试代理配置

### 问题 3：聊天无法发送
**原因**：后端服务未启动
**解决**：检查 FastAPI 是否运行在 http://127.0.0.1:8000

📖 **更多问题**？查看 [TESTING_GUIDE.md](TESTING_GUIDE.md) 中的"常见问题排查"
